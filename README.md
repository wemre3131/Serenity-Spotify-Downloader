# Serenity Exportify Downloader 🎵

A sleek, dark-themed GUI for converting Exportify Spotify CSVs into local MP3 files using `yt-dlp` and `FFmpeg`.

## ✨ Features
- **Dark UI:** Designed with the Serenity Spa palette (Gold/Sage/Charcoal).
- **Batch Processing:** Handles entire playlists from Exportify CSV exports.
- **MP3 Conversion:** Automatically converts video streams to high-quality 192kbps MP3s.

## 🛠️ Prerequisites
You must have **FFmpeg** installed and added to your system PATH.

## 🚀 Installation
1. Clone the repo:
   `git clone https://github.com/wemre3131/Serenity-Spotify-Downloader.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the app:
   `python downloader.py`
Or use the (Website)[serenity-downloader.vercel.app]
